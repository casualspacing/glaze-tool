⑤ 착색제 실험
같은 베이스에서 착색제를 여러 조합으로 실험하고 싶을 경우 착색제 실험에 추가 버튼을 클릭하면 착색제 실험 플래너로 데이터가 넘어갑니다. 여러 레시피를 한번에 넘기고 싶을 경우 전체 착색제 실험 버튼을 클릭하면 넘어갑니다.

재료명과 숫자가 같은 줄이거나 따로 줄이 나뉘어 있어도 인식합니다.
▶ 사용 방법

① 데이터 가져오기
배치 계산기에서 착색제 실험에 추가 또는 전체 착색제 실험 버튼을 클릭하면 레시피가 행으로 추가됩니다.

② 행 복제
같은 베이스로 여러 착색제 조합을 실험하려면 복제 버튼으로 행을 추가하세요. 이름을 변경해 구분할 수 있습니다.
예: SM-JB1 (원본), SM-JB2 (산화철 Red Iron Oxide 4g), SM-JB3 (산화니켈 Nickel Oxide 0.4g + 루틸 Rutile 1g)

③ 착색제 검색 및 용량 입력
상단 착색제 열 헤더에서 재료를 검색해 추가하세요. 영어·한글 모두 검색 가능합니다. 각 셀에 용량(g)을 입력하면 베이스 기준 건조 분말 그람이 자동 계산됩니다.
예를 들어 레시피 100g 기준으로 SM-JB2에는 산화철 4g, SM-JB3에는 산화니켈 0.4g + 루틸 1g을 넣어본다고 가정해볼게요. 베이스를 50g만 만들었다고 해도 걱정하지 마세요. 착색제 양이 자동으로 비례 계산됩니다!
예: SM-JB1 (원본), SM-JB2 (산화철 Red Iron Oxide 4g), SM-JB3 (산화니켈 Nickel Oxide 0.4g + 루틸 Rutile 1g)

착색제 검색해 추가 (영어/한글 모두 가능)
착색제 추가 후 용량 입력
니켈(한글 검색) 검색해 추가
루틸(영어 검색) 검색해 추가

④ 결과 복사
착색제 실험 세트를 완성했어요. 결과 복사 버튼을 누른 뒤 엑셀, 노션에 붙여넣기 할 수 있습니다.
